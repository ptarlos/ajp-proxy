AJP Tester

The purpose of this web application is to test servers via the AJP protocol. The Apache Tomcat Connectors (AJP) protocol is typically used between Apache LB and Tomcat (Java) servers.

Commands:

AJP CPing - Check that the AJP Server is Up
Syntax:
/ajpCPing?ip=<ajp.server.ip>&port=<ajp.server.port>
  • ajp.server.ip = AJP server IP address
  • ajp.server.port = AJP server port, default is 8009
Example:
http://localhost:9090/ajpCPing?ip=10.10.10.1&port=8009
AJP GET - Send GET message to AJP Server; browse a web-page
Syntax:
/ajpGet?ip=<ajp.server.ip>&port=<ajp.server.port>&query=<query.path>
  • ajp.server.ip = AJP server IP address
  • ajp.server.port = AJP server port, default is 8009
  • query.path = query path on the web server
Example:
http://localhost:9090/ajpGet?ip=10.10.10.1&port=8009&query=/mypage/test
